Medida que indica la sensibilidad de un ratón. A mayor DPI, el cursor se mueve más rápido y preciso con menos desplazamiento físico.

8. Single Sign-On (SSO)

Sistema de autenticación que permite acceder a múltiples aplicaciones o servicios con una sola cuenta y contraseña.

9. Sintaxis

Conjunto de reglas que determinan cómo deben estructurarse las instrucciones en un lenguaje de programación.

10. Identación

Espaciado al inicio de una línea de código que ayuda a organizar visualmente la estructura de un programa.

Único que identifica un sitio web en internet, como www.ejemplo.com.

12. Prompt

Instrucción que se ingresa en un sistema para que ejecute una acción. En IA, es la entrada que guía a los modelos de lenguaje.

13. Text box

Cuadro de texto dentro de una interfaz gráfica en el que el usuario puede escribir información.

14. ComboBox

Elemento de interfaz que combina un cuadro de texto con una lista desplegable de opciones seleccionables.

15. Ventana de contexto

Menú emergente que aparece al hacer clic derecho, mostrando opciones relacionadas con el objeto seleccionado.

16. Procesadores de lenguaje natural (NLP)

Tecnologías que permiten a las computadoras analizar, comprender y generar lenguaje humano.

17. Metadato

Dato que describe a otros datos. Ejemplo: en una fotografía, el metadato puede ser la fecha de captura o la ubicación.

18. Endpoint

Punto de acceso en una API o red donde se reciben o envían solicitudes y respuestas.

19. FastAPI (Tiangolo)

Framework de Python creado por Sebastián Ramírez para construir APIs rápidas, seguras y con documentación automática.

20. Partición de disco duro

División lógica de un disco físico en varias secciones independientes...

21. Shadows

Efectos visuales aplicados en interfaces gráficas para simular sombras y dar sensación de profundidad o realismo.

22. Mockup

Prototipo visual que representa cómo se verá una aplicación o sitio web antes de ser desarrollado.

23. Compilar

Proceso de traducir el código fuente escrito por el programador a un lenguaje que la computadora pueda ejecutar.

24. GUI (Graphical User Interface)

Interfaz gráfica que permite interactuar con un sistema mediante elementos visuales como botones, ventanas y menús.

28. LLM (Large Language Model)

Modelo de inteligencia artificial entrenado con grandes cantidades de texto para comprender y generar lenguaje natural.

29. Stack Overflow

Plataforma de preguntas y respuestas donde programadores de todo el mundo comparten soluciones y conocimiento.

30. ISP (Internet Service Provider)

Proveedor de servicios de internet encargado de ofrecer conexión a la red a hogares y empresas.

Framework de desarrollo web que proporciona componentes prediseñados y un sistema de cuadrícula para crear sitios responsivos rápidamente.

32. Materialize

Framework de CSS basado en Material Design de Google, que facilita la creación de interfaces limpias y modernas.

Lenguaje de marcado utilizado para estructurar el contenido de páginas web mediante etiquetas.

Lenguaje de hojas de estilo que define la apariencia de un sitio web: colores, fuentes, tamaños y disposición de los elementos.

35. JavaScript (JS)

Lenguaje de programación utilizado en la web para dotar de interactividad a las páginas, como animaciones o validación de formularios.

Lenguaje de programación del lado del servidor diseñado para el desarrollo web, muy utilizado en sitios dinámicos y gestores de contenido como WordPress.

40. SQL (Structured Query Language)

Lenguaje de consulta estructurado que permite gestionar y manipular bases de datos relacionales.

41. Base de datos

Conjunto organizado de información que puede ser almacenada, consultada y gestionada de forma digital.

42. Servidor

Computadora o sistema que proporciona servicios, recursos o datos a otros dispositivos llamados clientes.

43. Cliente

Es quien solicita servicios o información a un servidor dentro de una red.

44. Red LAN (Local Area Network)

Red de área local que conecta dispositivos en un espacio reducido, como una oficina o una casa.

45. Red WAN (Wide Area Network)

46. Router

Dispositivo que dirige el tráfico de datos entre redes y permite la conexión a internet.

47. Switch

Dispositivo de red que conecta múltiples dispositivos dentro de una LAN, gestionando el envío de datos.

48. Firewall

Dispositivo o software de seguridad que controla y filtra el tráfico de red, protegiendo contra accesos no autorizados.

49. VPN (Virtual Private Network)

Tecnología que crea una conexión segura y privada a través de internet, enmascarando la dirección IP real del usuario.

Identificador único que se asigna a cada dispositivo conectado a una red. Puede ser estática o dinámica.

Versión del protocolo IP más utilizada históricamente, que emplea direcciones de 32 bits (ejemplo: 192.168.0.1).

52. IPv6

Nueva versión del protocolo IP que usa direcciones de 128 bits, creada para suplir la escasez de direcciones IPv4.
